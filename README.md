# Identifizierung-der-Kreditausfallwahrscheinlichkeit-von-Kundensegmenten
Dieses Projekt konzentriert sich auf die Entwicklung eines datengestützten Entscheidungssystems zur Identifizierung von Kreditausfallwahrscheinlichkeiten in verschiedenen Kundensegmenten. Ausgehend vom Datensatz führe ich zunächst ein komplexes Data Cleaning und Feature Engineering durch, um aussagekräftige Variablen für die Kreditwürdigkeit zu generieren. Zur Modellierung vergleiche ich die Performance von logistischen Regressionen mit Random-Forest-Algorithmen. Die Güte der Modelle wird mittels ROC-Analysen und AUC-Werten evaluiert, um die Trennschärfe zwischen solventen und ausfallgefährdeten Kunden sicherzustellen. Die Ergebnisse liefern klare demographische Profile, etwa hinsichtlich Alter und Familienstand, die signifikant mit dem Zahlungsverhalten korrelieren und somit eine präzisere Risikobewertung im Kreditgeschäft ermöglichen.
